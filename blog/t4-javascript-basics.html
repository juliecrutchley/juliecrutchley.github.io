How does Javascript compare to HTML and CSS?

Javascript is also a 'web programming language' like HTML and CSS. However, HTML and CSS are used for structuring content and applying visual styles. Javascript can make websites, applications and games come alive! With Javascript, web pages become responsive to user interaction, e.g. performing a task after an element is clicked.

Explain control flow and loops using an example process from everyday life

Control flow refers to the way Javascript statements and loops are processed. By using if / else / while statements and loops you can create a controlled flow of processes, defined by logic, to achieve a result.

To explain, let's use the literal everyday life process of making your toast in the morning:

WHILE time=breakfast AND (you=hungry1 OR you=hungry2),
--->IF pantry contains bread AND best before date = today or later,
--->--->IF you=hungry1, add 1x toast to toaster,
--->--->ELSE add 2x toast to toaster,
--->ELSE have something else for breakfast

Explain the difference between accessing data from arrays and objects

Arrays are variables within which we can store several pieces of data, like a list.
You can access the items within an array by calling the array with bracket notation. Note arrays use zero-based indexing.

myBag = ["glasses", "phone", "mints"];

myBag[2]; //will return "mints"

Objects are similar to arrays except instead of using indexes to access their data, you can access their data through their properties using either bracket notation or dot operator.

var myBag = {
prop1: "glasses",
prop2: "phone",
prop3: "mints"
};

myBag["prop3"]; //will return "mints"

myBag.prop3; //will return "mints"

Explain what functions are and why they are useful

If we have some code we would like to run often, rather than writing the same or similar code every time, we can create a function with a logical name, say "checkIfPrimeNumber" and call it whenever we need. Every time we need to check if a number is a prime, we could call the function with the number as an argument, and get a true or false result.
