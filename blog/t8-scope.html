What is scope and how does it work in Javascript?

To explain Javascript scope, I like to use the analogy of an apartment building.

Within Javascript there are variables that are global, meaning they can be accessed by the entire program, and then there are variables that are local, meaning they were created within a function and can only be accessed within that function.
Everyone who lives in the apartment building has access to the building.
However, only I have access to my apartment.
This translates well to Javascript because my apartment can only be accessed by me ('local'), however everyone can access the building ('global').
