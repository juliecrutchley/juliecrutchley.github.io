Explain a time you got blocked on a simple problem

What was the problem?
The exercise was to create a program which would print an 8x8 grid chessboard, using #s for black squares and blank spaces for white squares.

What problem solving techniques did you use?
I used several problem solving techniques to solve this issue as I hadn't quite mastered nested loops at the time!

How did you feel throughout the process?
I felt very frustrated as I knew the program to write should be very simple. I also felt worried that I must have skimmed over the previous learning exercises and resources which explained the related concepts.

What did you learn?
I learned that it's OK to not know everything when you're just starting out, and that you will eventually figure it out! I also learned that, at least for me, waliking away from the exercise I am stuck on and returning the next day with a clear head, works really well!

What problem solving techniques did you use?
I used several problem solving techniques including pseudocoding, console.logging and good ol' Googling.

Reflect on how confident you feel using these problem solving techniques and processes:

Pseudocode
I have used this technique several times so far and it seems to work well for me. If I am approaching a complex exercise, I will begin by constructing a list of all the elements I think will need to be involved, i.e. loops, new lines, named variables, and then break the exercise down into small pieces of pseudocode to work on.

Trying something
Sometimes tutu-ing with random things can be deceivingly helpful! Often if I think my code looks fairly correct, I will just change one small random part of the code and see what results.

Rubber ducky method
Yes, I have used this method! I don't have a rubby ducky though, but the plant in my study suffices...

Reading error messages
This is also very helpful!

Console.logging
Again, this method is very helpful.

Googling
Google is a great resource to use when learning to code. Every time I have resorted to having to google, the issue has been solved very quickly. It's also very reassuring to see that others have googled the same issue.

Asking your peers and coaches for help
I haven't needed to do much of this yet, but I definitely will if I need to.

Improving your process with reflection
This is very important, sometimes if I am reading or watching something, I will write down in my own words what I have just learnt.

Explain these functions in your own words:

.map()
This method is used to take an existing array, call a function to do something on each of the items within that array, and create a new array with the results.

.filter()
This method is used to take an existing array, apply a filter such as 'only those items starting with letter A' and create a new array with the results.

.reduce()
This method reduces several numerical values in an array into one value.
