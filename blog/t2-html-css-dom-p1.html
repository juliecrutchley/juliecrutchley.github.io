Can you think of an analogy to describe HTML, CSS and the DOM?

Let's use the analogy of an outfit... bear with me and I apologise if this is wrong, the DOM is still unique to me!

The HTML would be the clothing items that make up the outfit, i.e: dress, belt, shoes.

The CSS would specify what those items will look like), i.e:
- The dress will be yellow and knee length
- The belt will be blue

The DOM (Document Object Model) would be a drawing of the outfit much like the one to the left.

What is meant by boxifying design?

Everything you see on a webpage is within a box. Boxifying design is the process of breaking down a proposed design into the boxes required and beginning to build the website from there.

What is the box model?

All the HTML content is by default already in boxes. In CSS, the Box Model is a reference to the way of looking at a design element by considering the total box contains margins, borders, padding and then the actual content.
By first considering this, you can then apply the correct design rules.
