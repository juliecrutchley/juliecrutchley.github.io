In the Javascipt Language, describe all the use cases for each of:

Parentheses ( )
- contain parameters being passed to functions
- contain conditional statements
- group expressions in order of execution
- execute a function

Brackets [ ]
- declare an array
- access an array element, an object property or a string character using bracket notation

Braces { }
- declare an object
- contain function code

Single quotes ' ' and Double quotes " "
- can be used interchangeably to contain a string, as long as they match
